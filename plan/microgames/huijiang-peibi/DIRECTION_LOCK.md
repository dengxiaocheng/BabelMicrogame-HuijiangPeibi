# Direction Lock: 灰浆配比

## One Sentence
玩家在材料短缺和赶工压力下调配灰浆，决定墙体是稳固、开裂，还是被暂时遮掩。

## Core Loop
1. 执行核心循环：查看今日材料 -> 调配灰浆 -> 搅拌 -> 涂抹墙段 -> 观察表面反馈 -> 抽检或下一墙段
2. 按 20 分钟节奏推进：正常配比 -> 材料不足 -> 雨水/干燥/催工 -> 抽检遮掩或返工

## Must Keep
- 核心机制必须保持：比例调配 + 搅拌时间 + 隐藏质量
- 核心循环必须保持：查看今日材料 -> 调配灰浆 -> 搅拌 -> 涂抹墙段 -> 观察表面反馈 -> 抽检或下一墙段
- 20 分钟结构只作为节奏，不扩成大项目

## Must Not Add
- 不做完整建筑模拟；核心是配比误差如何变成质量风险
- 不新增第二套主循环
- 不把小游戏扩成长期经营或开放世界

## Required State
- resource
- pressure
- risk
- relation
- round

## Success
在限定时长内完成主循环，并稳定进入至少一个可结算结局

## Failure
关键状态崩溃，或在本轮主循环中被系统淘汰
