# SCENE_INTERACTION_SPEC: 灰浆配比

## Scene Layout

```
+--------------------------------------------------+
|  状态栏：wall_quality / inspection_risk / 阶段    |
+--------------------------------------------------+
|                    |                              |
|   搅拌槽区域       |    墙段显示区域               |
|   (操作区)         |    (反馈区)                   |
|                    |                              |
+--------------------------------------------------+
|  材料余量          |   稠度仪表                    |
|  water: ████░░     |   [==========|---]           |
|  sand:   ██████░   |                              |

|                    |   [======|--------]           |
+--------------------------------------------------+
|  水滑杆: [=====|=====]  数值: 25                   |
```

## Scene Objects 与交互映射

| 场景对象 | 交互方式 | 影响 State |
|---------|---------|-----------|
| 水滑杆 | 拖拽调整数值 | water 消耗量 |
| 砂滑杆 | 拖拽调整数值 | sand 消耗量 |
| 灰滑杆 | 拖拽调整数值 | lime 消耗量 |
| 搅拌按钮 | 按住开始，松开停止 | stir_quality |
| 涂抹按钮 | 单击触发涂抹 | wall_quality, inspection_risk |
| 搅拌槽 | 显示搅拌动画和当前稠度 | 可视反馈 |
| 墙段显示 | 显示墙面纹理变化 | 可视反馈 |

## Feedback Channels

| 反馈元素 | 显示时机 | 反映 State |
|---------|---------|-----------|
| 稠度条 | 调整滑杆时实时更新 | 当前配比的稠度评估 |
| 搅拌动画 | 按住搅拌时播放 | 搅拌进度 |
| 墙面纹理 | 涂抹后更新 | wall_quality |
| 材料余量条 | 每次调配后更新 | water, sand, lime |
| 抽检风险条 | 每次涂抹后更新 | inspection_risk |
| 表面裂纹特效 | 墙段质量 < 50 时显示 | wall_quality |

## Input → State 追溯规则

1. 滑杆调整 → 配比计算 → deviation → 本批次质量系数
2. 按住时长 → stir_quality → 本批次质量系数
3. 涂抹按钮 → 本批次质量 × 累积 wall_quality → 新 wall_quality
4. 每次涂抹 → 检查 quality → 更新 inspection_risk
5. 每面墙段完成 → 按 inspection_risk 概率触发抽检

## Forbidden UI

- 不允许只给三种配方按钮代替滑杆
- 不允许用纯文字选择代替搅拌交互
- 不允许扩成完整建筑模拟（不显示整栋楼、不加楼层系统）
- 不允许添加需要后端 API 的功能

## Acceptance Rule

- 首屏必须让玩家看到：三个滑杆、搅拌按钮、材料余量
- 玩家操作滑杆必须产生稠度条实时变化
- 玩家按住搅拌必须产生搅拌动画
- 玩家涂抹后必须看到墙面视觉变化和数值更新
- 所有反馈能追溯到 Required State 中的至少一个
- 不得只靠随机事件文本或普通选择按钮完成主循环
