# 灰浆配比

Babel microgame — 玩家在材料短缺和赶工压力下调配灰浆，决定墙体是稳固、开裂，还是被暂时遮掩。

## 在线试玩

https://dengxiaocheng.github.io/BabelMicrogame-HuijiangPeibi/

## 本地运行

```bash
npm install
npm run dev
```

浏览器打开 Vite 提示的本地地址即可。

## 测试

```bash
npm test
```

## 构建

```bash
npm run build
```

构建产物输出到 `dist/`，可直接部署到任意静态托管服务。
